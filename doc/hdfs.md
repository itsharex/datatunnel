### Reader 参数

| 参数key            | 是否必填  | 默认值    |描述                                  |
| :-----            | :-----   | :------  | :------                             |
| path              | √        |          | 读取hdfs文件路径，路径前缀必须为：/user/datawork/users/{当前登录账号}        |
| fileNameSuffix    |          |          | 文件名后缀，例如只读取csv文件: .csv       |
